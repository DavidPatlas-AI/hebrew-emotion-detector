## 📁 מבנה התיקיות

### 📊 **מחקרי שוק**
**מיקום:** `מחקרי שוק/`

**מיועד עבור:**
- מחקרי שוק על תחום זיהוי הרגשות
- ניתוח מתחרים ופתרונות קיימים
- מחקרי משתמשים וצרכי שוק
- דוחות מגמות בתחום הטכנולוגיה הטיפולית
- סקרי לקוחות ומשוב משתמשים

**פורמטים מומלצים:**
- PDF, DOC, DOCX
- Excel/CSV עם נתונים
- PowerPoint עם מצגות
- תמונות של גרפים ותרשימים

**סוגי קבצים:**

## 🔍 איך להשתמש בתיקייה

### שלב 1: העלאת קבצים
1. **בחר את התיקייה המתאימה** לסוג התוכן שלך
2. **העלה את הקבצים** לתיקייה הרלוונטית
3. **תן שמות ברורים** לקבצים (תאריך + תיאור)
4. **הוסף הערות** אם נדרש

### שלב 2: בקשת ניתוח
כשתרצה שאני אעבור על החומרים:
1. **ציין איזה קבצים** לבדוק
2. **הסבר מה המטרה** של הניתוח
3. **בקש סוג עבודה ספציפי** (סיכום, רעיונות, פיתוח)

### שלב 3: קבלת תוצאות
אני יכול לעזור לך:
- 📖 **לסכם ולנתח** מחקרים
- 💡 **להציע רעיונות** חדשים
- 🔧 **לפתח כלים** חדשים
- 📊 **לנתח נתונים** ולהפיק תובנות
- 🚀 **לתכנן אסטרטגיות** פיתוח

## 🎯 דוגמאות לשימוש

### דוגמה 1: מחקר שוק
```
"שמתי מחקר על מתחרים בתיקיית מחקרי שוק. 
אתה יכול לעבור עליו ולתת לי:
1. סיכום של 3 התחרים העיקריים
2. זיהוי פערים בשוק
3. רעיונות לפיצ'רים ייחודיים"
```

### דוגמה 2: כלי חדש
```
"יצרתי פרוטוטיפ של אפליקציה בתיקיית כלים חדשים.
תוכל לבדוק את הקוד ולהציע:
1. שיפורים טכניים
2. תכונות נוספות
3. אופטימיזציה של הביצועים"
```

### דוגמה 3: רעיון חדש
```
"יש לי רעיון למוצר חדש שתיעדתי ברעיונות חדשים.
אשמח שתעזור לי:
1. לפתח את הרעיון לפלאן מפורט
2. לזהות אתגרים טכניים
3. ליצור POC ראשוני"
```
